# Changelog

## v1.0.0 — 2026-06-08

Primeira versão pública da skill `nem-parece-ia`.

Esta versão adapta a lógica editorial do projeto `stop-slop` para português do Brasil, com foco em remover marcas de escrita artificial, jargão corporativo, falsa profundidade, estruturas performáticas e cadência robótica.

| Área | Entrega |
|---|---|
| Núcleo | `SKILL.md` com gatilhos, fluxo de revisão, rubrica e formato de resposta recomendado. |
| Frases | Lista de aberturas, metacomentários, intensificadores, jargões e fórmulas de LinkedIn em PT-BR. |
| Estruturas | Diagnóstico de contraste fabricado, pergunta performática, falsa agência, passiva evasiva e abstração vaga. |
| Ritmo | Regras para tríades automáticas, punchlines, fragmentação dramática e tradução mental do inglês. |
| Registro | Calibração para post, ensaio, e-mail, institucional, técnico, acadêmico, comercial e criativo. |
| Exemplos | Modelos antes/depois em português brasileiro. |
| Uso responsável | Nota de atribuição e diretriz explícita contra uso para enganar detectores ou ocultar autoria. |
