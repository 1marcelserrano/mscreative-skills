# Estruturas pt-BR a revisar

Usar esta referência quando o problema do texto não estiver apenas em palavras soltas, mas na arquitetura da frase, do parágrafo ou do argumento.

## Contraste fabricado

Textos de IA e posts performáticos usam contraste para fabricar insight. Manter contraste quando houver oposição real. Cortar quando ele só cria suspense.

| Padrão | Problema | Revisão |
|---|---|---|
| Não é sobre X. É sobre Y. | Cria profundidade automática | Afirmar Y diretamente |
| Não se trata de X, mas de Y | Fórmula de reframe | Usar quando a distinção for necessária; caso contrário, cortar X |
| X não é o problema. Y é. | Pivot previsível | Explicar Y como problema |
| Parece X, mas na verdade é Y | Suspense artificial | Declarar Y e mostrar evidência |
| A pergunta não é X. A pergunta é Y. | Dramatiza a tese | Fazer a pergunta certa apenas se ela orientar o texto |
| Deixa de ser X e passa a ser Y | Arco falso de transformação | Nomear a mudança concreta |
| Não é apenas X, é também Y | Hedge aditivo | Se Y importa, integrar Y sem teatro |

## Listagem negativa

Listar o que algo não é antes de dizer o que é costuma alongar sem ganhar precisão.

| Padrão | Revisão |
|---|---|
| Não é uma ferramenta. Não é uma metodologia. É uma mudança de postura. | “É uma mudança de postura.” Depois provar. |
| Não foi sorte. Não foi timing. Foi consistência. | “A consistência produziu o resultado.” |
| Não é rápido. Não é fácil. Mas funciona. | Explicar condição, custo e resultado. |

## Pergunta performática

Perguntas retóricas podem envolver o leitor, mas viram muleta quando o texto responde imediatamente com obviedade.

| Padrão | Problema | Revisão |
|---|---|---|
| E se eu te dissesse que...? | Fórmula de revelação | Afirmar o ponto |
| Já parou para pensar que...? | Tom professoral | Entrar no argumento |
| Sabe o que isso significa? | Suspense barato | Dizer a consequência |
| O que isso nos ensina? | Moral pronta | Mostrar aprendizado específico |
| Por que isso importa? | Pode ser útil, mas frequentemente redundante | Responder se a seção realmente depender da pergunta |

## Falsa agência

Coisas abstratas não decidem, não querem, não pedem e não enxergam. Pessoas, equipes, clientes, instituições e sistemas fazem ações. Nomear o agente aumenta responsabilidade.

| Padrão | Problema | Revisão |
|---|---|---|
| O mercado exige X | Quem exige? Clientes, compradores, reguladores? | “Clientes enterprise recusam propostas sem X.” |
| Os dados mostram X | Dados não mostram sozinhos; alguém interpreta | “A análise dos dados indica X.” |
| A estratégia pede X | Estratégia não pede | “A liderança priorizou X.” |
| A cultura mudou | Cultura muda por comportamento | “A equipe passou a recompensar X.” |
| A conversa avançou | Conversa não anda sozinha | “O grupo passou a discutir X.” |
| A decisão surgiu | Decisões não surgem | “O comitê decidiu X.” |
| A tecnologia permite | Pode ser aceitável, mas genérico | “O modelo X reduz Y e viabiliza Z.” |

### Atribuição vaga

Opinião atribuída a uma autoridade genérica, sem fonte. Nomear quem, onde e quando, ou cortar a atribuição.

| Padrão | Revisão |
|---|---|
| Especialistas afirmam que... | “Em estudo de 2024, o pesquisador X mostrou que...” |
| Estudos apontam que... | Citar o estudo, o ano e o achado |
| Segundo relatórios do setor... | Nomear o relatório e a entidade |
| Observadores notam que... | Cortar ou nomear quem observou |
| É amplamente reconhecido que... | Mostrar a base ou cortar |

## Cópula inflada

LLM troca o verbo simples por uma construção cerimoniosa. A frase fica mais longa e diz o mesmo.

| Evitar | Preferir |
|---|---|
| serve como | é |
| funciona como | é, faz |
| se posiciona como | é, parece |
| se apresenta como | é |
| representa um / marca um | é, faz |
| desempenha um papel em | tem papel em |
| ostenta / apresenta / oferece (genérico) | tem, dá |
| atua como | é |

**Regra:** se o sujeito simplesmente *é* aquilo, escrever “é”. A cópula inflada só se justifica quando há diferença real entre ser e funcionar como.

## Gerúndio de falsa profundidade

LLM cola um gerúndio no fim da frase para simular análise. O gerúndio repete a frase anterior em tom de conclusão, sem acrescentar fato.

| Padrão | Problema | Revisão |
|---|---|---|
| ..., destacando sua importância. | Repete que é importante sem provar | Cortar o gerúndio ou nomear a consequência |
| ..., refletindo tendências mais amplas. | Generaliza sem dado | Nomear a tendência e a fonte, ou cortar |
| ..., pavimentando o caminho para X. | Promessa vaga de futuro | Dizer o que X muda agora |
| ..., abrindo portas para novas possibilidades. | Clichê de fechamento | Cortar |
| ..., consolidando seu papel como Y. | Inflação de papel | Cortar ou dizer o que Y faz |
| ..., demonstrando seu compromisso com Z. | Atribui intenção sem prova | Mostrar a ação concreta |

**Teste:** se o gerúndio puder virar ponto final na palavra anterior sem perda, cortar.

## Passiva evasiva

A voz passiva não é proibida. Ela é problema quando esconde responsabilidade ou enfraquece ação. Em texto técnico ou institucional, pode ser necessária para focar processo, norma ou resultado.

| Padrão | Revisão preferencial |
|---|---|
| Foi decidido que | Quem decidiu? |
| Erros foram cometidos | Quem errou? |
| A proposta foi rejeitada | Quem rejeitou e por qual critério? |
| O projeto será implementado | Quem implementará e quando? |
| Medidas foram tomadas | Quais medidas, por quem? |

## Abstração sem lastro

Substantivos abstratos podem ser úteis, mas viram névoa quando substituem fatos.

| Abstração vaga | Pergunta que força concretude |
|---|---|
| transformação | O que muda no comportamento, processo ou resultado? |
| inovação | O que é novo e para quem? |
| impacto | Em qual métrica, pessoa ou sistema? |
| complexidade | Qual variável torna o problema difícil? |
| oportunidade | Quem ganha o quê se agir? |
| desafio | Qual bloqueio impede avanço? |
| maturidade | Que prática comprova maturidade? |
| alinhamento | Quem concorda com qual decisão? |

### Teste de imagem

Antes de manter uma frase abstrata, tentar desenhar a cena. Se não dá pra desenhar, reescrever até dar.

| Abstrato (sem imagem) | Concreto (dá pra desenhar) |
|---|---|
| As camadas de critério que existem antes da execução | A regra que você aplica antes de abrir o arquivo |
| A maturidade do processo aumentou | A equipe parou de refazer o mesmo passo três vezes |
| Há um desalinhamento estrutural | Duas áreas usam planilhas diferentes para o mesmo número |

Receita: substantivo concreto, verbo de ação, exemplo. Se só sobra a categoria abstrata, a frase ainda não está pronta.

## Narrador distante

Evitar voz que observa tudo de cima e não coloca ninguém na cena.

| Padrão | Problema | Revisão |
|---|---|---|
| As pessoas tendem a... | Generalização frouxa | “Quando você..., tende a...” ou caso específico |
| Ninguém percebe que... | Autoridade falsa | “Muitas equipes ignoram...” se houver base |
| Todo mundo sabe que... | Absoluto preguiçoso | Nomear grupo ou evidência |
| Isso acontece porque... | Pode soar professoral | Ir direto à causa concreta |
| A sociedade caminha para... | Abstração grande demais | Nomear atores, comportamentos e sinais |

## Variação elegante

LLM evita repetir um nome trocando por sinônimo a cada menção. O leitor perde o fio de quem é quem. Repetir o nome é mais claro que variar.

| Antes | Depois |
|---|---|
| A empresa cresceu. A companhia contratou. A organização então decidiu. A firma anunciou. | A empresa cresceu, contratou e anunciou. |
| Maria propôs o plano. A executiva defendeu a ideia. A gestora então apresentou. | Maria propôs o plano, defendeu a ideia e apresentou. |

**Regra:** repetir o nome próprio ou o substantivo exato não é erro. Variar só para fugir da repetição cria névoa. Pessoa tem um nome; usar ele.

## Falso intervalo

LLM usa “de X a Y” para parecer abrangente, mesmo quando X e Y não formam escala.

| Padrão | Problema | Revisão |
|---|---|---|
| Da tradição ancestral à inovação de ponta | X e Y não são extremos de um eixo | Listar o que o texto cobre de fato |
| Do micro ao macro, do local ao global | Par genérico de oposições | Nomear os casos reais |
| Desde startups até grandes corporações | Pode ser real; checar se há meio | Manter só se o intervalo tiver pontos no meio |

**Teste:** existe um meio real entre X e Y? Se o intervalo é só par de opostos para soar completo, listar os itens concretos.

## Conclusão automática

Textos de IA concluem com síntese genérica e motivacional. Trocar por consequência, decisão ou fechamento informativo.

| Evitar | Preferir |
|---|---|
| No fim, tudo se resume a... | Fechar com a consequência específica |
| O futuro será de quem... | Fechar com decisão ou critério |
| A escolha é sua | Fechar com ação concreta, se houver call to action |
| E talvez essa seja a maior lição | Dizer a lição sem suspense |
| Mais do que nunca, precisamos... | Nomear quem precisa fazer o quê agora |

## Seção de desafios formulaica

LLM fecha textos com uma seção de “desafios e perspectivas” que não acrescenta fato.

| Padrão | Revisão |
|---|---|
| Apesar de seus avanços, X enfrenta desafios típicos do setor. | Nomear o desafio específico, com data ou número. |
| Apesar dos desafios, X continua a prosperar. | Dizer o que X fez em resposta, e o resultado. |
| Olhando para o futuro, as perspectivas são promissoras. | Cortar ou citar o plano concreto e o prazo. |

**Regra:** se a frase serve para qualquer assunto, ela não pertence a este. Trocar por fato datado.

## Critério final

Quando a estrutura parecer pronta para qualquer assunto, reescrever. Uma boa frase deve carregar sinais do caso específico: agente, objeto, restrição, consequência, escala ou tensão real.
