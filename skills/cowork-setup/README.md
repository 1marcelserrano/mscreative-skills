# cowork-setup (corpo da skill)

Monta, organiza ou diagnostica um workspace do Claude Cowork.

Este diretório é a skill instalável. O agente carrega o `SKILL.md`.

Para a visão geral, instruções de instalação e exemplos, veja o [README da raiz](../../README.md).

## Arquivos

- `SKILL.md` — frontmatter + corpo carregado pelo Claude
- `README.md` — este arquivo
- `references/templates.md` — templates dos arquivos de contexto
- `evals/evals.json` — casos de trigger / não-trigger pra rodar no `prompt-eval`
